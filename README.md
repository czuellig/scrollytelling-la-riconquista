# La Reconquista - Scrollytelling 

![keyhublogo](/docpics/keyhub_logo_white.png)

IM5/Visua Projekt von Livia Vogt und Cédric Züllig. <br>
**La Reconquista** ist eine interaktive Scrollytelling-Website, die mit Bildern, Skizzen und prägnanter Typografie eine poetische Geschichte über Pflanzen im urbanen Raum erzählt. Inspiriert von Franz Hohlers Erzählung «Die Rückereroberung» lädt das Projekt dazu ein, die stille Kraft der Natur neu wahrzunehmen.
